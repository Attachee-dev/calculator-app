# calculator-app

Main  Features


Basic Arithmetic: Perform addition, subtraction, multiplication, and division.

Responsive Design: The layout is designed to look good on various screen sizes.

Clean UI: Simple color palette and a modern look.

Clear Functionality: The 'C' button clears the current calculation.

JavaScript Logic: All calculations are handled by a single script.js file, keeping the logic separate from the presentation.

 
 Technologies Used


HTML: The structure of the calculator and its buttons.

CSS: Styling for a clean, minimalist design. It uses a custom stylesheet as well as Bootstrap for some utility classes.

JavaScript: The core logic that handles user input, performs calculations, and updates the display.

 File Structure


The project consists of three main files:

index.html: The main HTML file that provides the structure of the calculator.

style.css: Contains all the CSS rules to style the calculator and give it a polished, minimalist look.

script.js: The JavaScript file that contains all the calculator's functionality, from handling button clicks to performing calculations.

 How to Use


Simply open the index.html file in your web browser. The calculator is fully functional and ready to use without any additional setup.
